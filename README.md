## 📌 Sobre o Projeto

Este repositório armazena a resolução da **Atividade 18** do curso do SENAI. O foco desta etapa é consolidar os fundamentos da **Lógica de Programação** por meio do desenvolvimento de algoritmos. 

Para garantir uma compreensão completa do fluxo de dados e da sintaxe básica, cada exercício foi resolvido de duas maneiras distintas:
1. **Fluxograma:** Criação de diagramas de blocos (Fluxogramas) utilizando o software **Flowgorithm**.
2. **Pseudocódigo:** Escrita de pseudocódigo (Portugol) utilizando o software **VisuAlg**.

---

## 💻 Exercícios Desenvolvidos

Abaixo está o detalhamento de cada um dos 7 algoritmos criados nesta atividade, abrangendo desde operadores matemáticos básicos até estruturas de decisão (condicionais).

| ID | Tema | Descrição do Algoritmo | Conceito Principal Aplicado |
| :---: | :--- | :--- | :--- |
| **01** | 🔢 **Soma** | Lê valores inseridos pelo usuário, realiza a adição e exibe o resultado. | *Operadores Aritméticos Básicos* |
| **02** | 📦 **Paralelepípedo** | Recebe as dimensões de um paralelepípedo e realiza o cálculo de sua área/volume. | *Lógica Matemática e Fórmulas* |
| **03** | 💰 **Comissão** | Calcula o valor da comissão de um funcionário com base em uma porcentagem sobre suas vendas. | *Cálculo de Porcentagem* |
| **04** | 🚗 **Dirigir** | Verifica a idade informada pelo usuário e determina se ele possui idade legal para dirigir. | *Estrutura Condicional Simples* |
| **05** | 🔐 **Senha** | Simula um sistema de login, verificando se a senha inserida corresponde à senha cadastrada. | *Operadores Relacionais / Validação* |
| **06** | 🎓 **Nota** | Lê as notas de um aluno, calcula a média e retorna o status (Aprovado ou Reprovado). | *Cálculo de Média e Condicionais* |
| **07** | 🗳️ **Votar** | Avalia a idade do usuário e informa as regras eleitorais (Não vota, Voto Facultativo ou Obrigatório). | *Estruturas Condicionais Aninhadas* |

## 🛠️ Tecnologias e Ferramentas

As seguintes ferramentas foram utilizadas para a construção e teste dos algoritmos:

* **[VisuAlg 2.5]:** Interpretador de pseudocódigo em português, ideal para o aprendizado inicial de programação. Arquivos com extensão `.alg`.
* **[Flowgorithm]:** Ferramenta gráfica que permite a criação e execução de algoritmos através de fluxogramas. Arquivos com extensão `.fprg`.

---

## ⚙️ Como executar os projetos na sua máquina

Para testar os códigos, siga os passos abaixo:

1. **Clone o repositório:**
   `git clone https://github.com/ThiagoMarcelinoVieira/atv18.git`

2. **Para rodar os Pseudocódigos (`.alg`):** Abra o programa **VisuAlg**, vá em `Arquivo > Abrir`, selecione o exercício desejado e pressione `F9` para executar.

3. **Para rodar os Fluxogramas (`.fprg`):** Dê um duplo clique no arquivo desejado (se o **Flowgorithm** já estiver instalado) ou abra o programa e use a opção de carregar arquivo. Clique no botão de "Play" verde para iniciar a execução.
